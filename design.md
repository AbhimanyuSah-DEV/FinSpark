# BankShield AI Design System

## Vision
Build **Bank of Maharashtra 2.0** — a modern digital banking platform with an integrated AI-powered Security Operations Center (SOC).

### Philosophy
- Backend = BrainCore (Intelligence)
- Frontend = StoryView (Visualization)
- Enterprise banking UI, not a fintech app.
- Follow Bank of Maharashtra branding while modernizing UX.

## Brand Identity

Design Rules:
- Large whitespace

- 12px radius
- Soft shadows
- Lucide React icons
- Responsive
